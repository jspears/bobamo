Bobamo!

 

This is a sample app that generates all the stuff needed to build a webapp with Backbone.js, Mers, Mongoose, requirejs, jquery, Backbone Forms and Twitter Bootstrap from a mongoose schema.

Learn more »

Use It

  1. install mongodb
  2. install npm
  3. #npm install bobamo
  4. start the app #bobamo
